## About the Course | عن الدورة
This course is designed for learners who want to master SPSS for data analysis.  
Each lesson contains hands-on examples and exercises to build practical skills.  

تم تصميم هذه الدورة للمتعلمين الراغبين في إتقان برنامج SPSS للتحليل الإحصائي.  
كل درس يحتوي على أمثلة عملية وتمارين لبناء مهارات عملية متقدمة.

All files are organized by lesson for easy access.  
تم تنظيم جميع الملفات حسب الدروس لتسهيل الوصول إليها.
